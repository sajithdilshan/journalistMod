Introduction

This is my first WordPress theme and I hope you would like it.

How to install

  1. Open up your FTP program and login to your server. Navigate to your WordPress folder and upload the entire folder (journalistMod) to your wp-content/themes folder.

  2. Login to WordPress administration normally and select Appearance -> Themes from the menu.

  3. You should see a freshly installed theme called The JournalistMod 0.4 and a few basic details about the theme.

  4. Activate this theme by clicking 'Activate' link below the theme description.

  5. View your site to see The JournalistMod in all its glory.

Note: A page refresh may be required.
