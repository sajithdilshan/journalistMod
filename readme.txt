How to install
-----------------

You should be looking at the The JournalistMod folder right now if your reading this; if not, unzip the archive. You should then see a folder called journalistMod. 

Open up your FTP program and login to your server. Navigate to your WordPress folder and upload the entire folder (journalistMod) to your wp-content/themes folder.

Login to WordPress administration normally and select Appearance -> Themes from the menu.

You should see a freshly installed theme called The JournalistMod 0.4 and a few basic details about the theme. 

Activate this theme by clicking 'Activate' link below the theme description.

View your site to see The JournalistMod in all its glory. 
	
Note: A page refresh may be required.
